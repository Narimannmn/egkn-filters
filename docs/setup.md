# Установка расширения «ЕГКН Фильтр»

Это пошаговая инструкция для обычного пользователя. Расширение работает в браузерах на базе Chromium: **Google Chrome**, **Microsoft Edge**, **Brave**, **Yandex Browser**, **Opera**. В **Safari** не работает.

---

## Шаг 1. Скачайте сборку

1. Откройте страницу [Releases](https://github.com/Narimannmn/egkn-filters/releases) этого репозитория.
2. У самой свежей версии в разделе **Assets** скачайте архив `egkn-filter-<версия>.zip`.
3. Распакуйте архив в любую удобную папку (например, `~/Downloads/egkn-filter`). Внутри должны лежать `manifest.json`, папка `assets/` и другие файлы — это и есть распакованное расширение.

> Если на странице релизов архива нет — соберите расширение сами по инструкции в [contributing.md](./contributing.md#сборка), либо попросите автора выложить сборку.

## Шаг 2. Откройте страницу расширений

| Браузер | Адрес                  |
| ------- | ---------------------- |
| Chrome  | `chrome://extensions`  |
| Edge    | `edge://extensions`    |
| Brave   | `brave://extensions`   |
| Opera   | `opera://extensions`   |
| Yandex  | `browser://extensions` |

Скопируйте адрес в адресную строку и нажмите Enter.

## Шаг 3. Включите режим разработчика

В правом верхнем углу страницы расширений включите переключатель **«Режим разработчика»** (Developer mode). Появится новая панель с кнопками **«Загрузить распакованное»**, **«Упаковать расширение»**, **«Обновить»**.

## Шаг 4. Загрузите распакованное расширение

1. Нажмите **«Загрузить распакованное»** (Load unpacked).
2. В диалоге выберите папку, в которую вы распаковали архив на шаге 1 (ту, где лежит `manifest.json`).
3. В списке появится карточка **«ЕГКН Фильтр»** с зелёным переключателем.

## Шаг 5. Проверьте работу

1. Откройте [https://map.gov4c.kz/egkn/](https://map.gov4c.kz/egkn/) и при необходимости авторизуйтесь.
2. В левом нижнем углу страницы должна появиться круглая синяя кнопка **🔍** — это и есть запуск расширения.
3. Нажмите на неё. Откроется панель «Фильтр ЕГКН» с выбором области/района, групп слоёв и фильтров.
4. Ваш активный район подставится автоматически. Выберите слой — расширение загрузит все участки и покажет их списком с фильтрами по статусу и виду права.

---

## Обновление

Когда выходит новая версия:

1. Скачайте свежий архив со страницы Releases и распакуйте **поверх** старой папки (или в новую — тогда удалите старое расширение в браузере).
2. На странице `chrome://extensions` нажмите кнопку обновления (круглая стрелка) у карточки «ЕГКН Фильтр».

## Удаление

На странице `chrome://extensions` нажмите **«Удалить»** на карточке расширения. Папку с распакованной сборкой можно удалить вручную.

## Частые проблемы

**Кнопка 🔍 не появляется.**
Проверьте, что вы открыли именно `https://map.gov4c.kz/egkn/...` (а не корневой `map.gov4c.kz`). Расширение работает только на путях `/egkn/*`. Если страница уже была открыта до установки — обновите её (Cmd/Ctrl + R).

**Панель открывается, но в списке «Область и район» нет вашего района.**
Расширение читает справочник районов с того же бэкенда ЕГКН. Если вашего района нет — проверьте, что вы залогинены, и что в самом портале ЕГКН он доступен.

**Загрузка участков долгая или зависла.**
Для районов с большим количеством участков (несколько тысяч) загрузка может занять до 10 минут. В заголовке списка отображается прогресс: «загружено N из M». Если счётчик не двигается больше минуты — закройте панель и откройте снова: расширение перезапросит данные.

**Браузер показывает предупреждение «Отключите расширения, работающие в режиме разработчика».**
Это стандартное поведение Chrome для распакованных расширений. Можно нажать «Оставить» — расширение продолжит работать. Если предупреждение мешает, спрячьте его кнопкой «×» — оно появится снова при следующем запуске браузера.

**Ничего не помогло.**
Откройте DevTools (F12) на странице `map.gov4c.kz/egkn/`, вкладка **Console**, и пришлите скриншот ошибки в [issues репозитория](https://github.com/Narimannmn/egkn-filters/issues).
